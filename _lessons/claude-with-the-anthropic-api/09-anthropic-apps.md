- Claude를 활용해 만들어진 대표 애플리케이션 형태를 이해하기
- Claude Code가 무엇이고 어떻게 쓰는지 파악하기
- Computer Use의 개념과 활용을 이해하기

이 모듈은 지금까지 배운 API 기능들(도구 사용, 에이전틱 루프)이 **실제 제품**에서 어떻게 쓰이는지 보여줘요.

## Claude Code

**Claude Code**는 터미널·IDE에서 동작하는 **에이전틱 코딩 도구**예요. 앞에서 배운 도구 사용 + 에이전틱 루프의 결정판이에요.

- 파일 읽기/쓰기/편집, 셸 명령 실행, 코드베이스 탐색을 **스스로** 수행해요.
- 자연어로 작업을 지시하면 여러 단계에 걸쳐 계획하고 실행해요.

### 셋업과 사용

```bash
npm install -g @anthropic-ai/claude-code   # 설치 예시
claude                                       # 프로젝트 폴더에서 실행
```

- 실행 후 자연어로 요청하면(예: "이 버그 고쳐줘", "테스트 추가해줘") Claude가 파일을 탐색·수정하고 명령을 실행해요.

### MCP 서버로 확장

Claude Code에 **MCP 서버**를 연결하면 능력이 확장돼요. 예를 들어 GitHub·DB·Slack MCP를 붙이면 그 시스템들과 직접 상호작용할 수 있어요 (앞 레슨의 MCP와 연결).

## Computer Use (컴퓨터 사용)

**Computer Use**는 Claude가 **화면을 보고(스크린샷) 마우스·키보드를 조작**해 GUI 애플리케이션을 직접 다루는 기능이에요.

```
스크린샷 촬영 → Claude가 화면 파악 → 클릭/입력 좌표 지시 → 내 환경이 실행 → 다시 스크린샷 → 반복
```

- API로 정형화되지 않은(전용 API가 없는) 웹·데스크톱 앱을 자동화할 때 유용해요.
- 클라이언트 측 도구예요 — **실행 환경(가상 데스크톱 등)은 내가 제공**하고, Anthropic은 스크린샷을 해석해 다음 행동을 지시해요.
- 아직 신중히 다뤄야 하는 기능이라, 격리된 환경에서 쓰고 민감한 동작엔 확인 단계를 두는 게 좋아요.

## 정리

- **Claude Code**: 터미널/IDE의 에이전틱 코딩 도구 — 파일·셸을 스스로 다루며, MCP로 확장.
- **Computer Use**: 스크린샷+마우스/키보드로 GUI를 직접 조작 (실행 환경은 내가 제공).
- 둘 다 앞에서 배운 **도구 사용 + 에이전틱 루프**가 실제 제품이 된 사례예요.
