## 실습: 나만의 프로젝트 고르기

이 레슨은 개념 설명이 아니라 **직접 하는 활동**입니다. 남은 강의 동안 계속 작업할 **중간 규모의 다단계 프로젝트**를 하나 고릅니다. 조건은:

- 여러 종류의 작업이 포함될 만큼 **충분히 실질적**일 것
- 약 1시간 안에 끝낼 수 있을 만큼 **감당 가능**할 것
- 정말로 만들고 싶거나 이루고 싶은, **진심으로 관심 있는** 것일 것

### 프로젝트 예시

- **커뮤니케이션 프로젝트** — 잘 아는 주제로 발표 자료 만들기, 복잡한 주제를 일반 청중에게 설명하는 글 쓰기, 아이디어 제안서·피치 작성, 이력/자기소개 작성
- **리서치 프로젝트** — 관심 주제를 조사·분석
- **창작 프로젝트** — 이야기의 캐릭터 개념 개발 등

## Delegation 적용하기

고른 프로젝트에 앞 레슨의 Delegation을 적용합니다.

- **Problem Awareness** — 이 프로젝트의 목표와 필요한 작업은 무엇인가?
- **Platform Awareness** — 어떤 AI가 어떤 부분을 잘 도울 수 있나?
- **Task Delegation** — 내가 직접 할 부분과 AI와 협업할 부분을 어떻게 나눌까?

## 핵심 정리

- 이 레슨은 강의 전체를 관통할 실습 프로젝트를 정하는 활동입니다.
- 실질적이면서 1시간 내 가능하고, 관심 있는 다단계 프로젝트를 고릅니다.
- Problem/Platform Awareness와 Task Delegation을 실제 프로젝트에 적용해봅니다.
